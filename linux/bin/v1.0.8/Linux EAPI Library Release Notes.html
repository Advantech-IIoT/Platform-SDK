Top of page

Overview

Linux EAPI Library currently include libEAPI，libATGPIO and libEAPI-IPS. All the ec library depends on the ec linux driver.

Top of page

Support

Linux EAPI Library depend on GLIBC 2.23 or above version, and GCC/G++ 5.4.0 or above version to compile is suggested.

Top of page

Known Issue

1. Current UPS library only supports PCB version 01-2(including) or above version. PCB version 01-1 (inclusive) may print "unknown reply" frequently.

2. Due to use the old EC firmware, UNO-1372G-E3AE LED operation was must be use the GPIO API .
