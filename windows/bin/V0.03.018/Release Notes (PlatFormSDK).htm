Important Features

Brightness

  • This tool is used to adjust the brightness of screen.

ETP

  • The tool is used to control the ETP data status.

GPIO

  • The tool is used to set four digital inputs and four digital outputs from GPIO pins for most of on/off trigger and status reading.

HWMonitor

  • This tool is used to get information of temperature and voltage.

LED

  • This tool is used to get all Available LED Status and set LED status.

RedundantPowerStatus

  • This tool is used to get Redundant Power status.
